============
Introduction
============

This extension implements a simple real estate structure based on OpenImmo. Properties can be easily created and displayed on the FrontEnd.

We offer an automatic `OpenImmo Importer <https://www.ifabrik.de/loesungen/typo3-extension-real-estate/>`__ which gets the exported properties and imports them into the TYPO3 CMS. For more information regarding the OpenImmo importer, contact us via email on info@ifabrik.de or visit our website: `OpenImmo Importer <https://www.ifabrik.de/loesungen/typo3-extension-real-estate/>`__.
